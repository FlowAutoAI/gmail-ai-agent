# Gmail AI Agent

**By FlowAuto AI**

## Overview
An intelligent AI agent that automates Gmail workflows. This tool utilizes the Gmail API and LLMs to parse email intent, categorize incoming messages, and draft intelligent responses automatically.

## Key Features
* **Automated Triage:** Sorts emails based on urgency and context.
* **Smart Drafting:** Generates draft responses for review using AI.
* **Inbox Organization:** Reduces manual sorting time.

* Python 3.10+
* Gmail API (Google Workspace)

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Add your `credentials.json` from Google Cloud Console.
4. Run the agent: `python gmail_bot.py`
