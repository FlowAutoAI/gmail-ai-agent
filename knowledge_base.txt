# BrickIntel Investment Tool - FAQ & Policies

1. WHAT IS IT: An AI-driven analytics platform that predicts the resale value of Lego sets.

2. ACCESS: This is a 100% FREE tool provided exclusively for subscribers of the BridgesBricks YouTube channel.

3. SCORING FACTORS: The "Investment Confidence Score" (out of 100) is calculated using these weighted metrics:
   - Retirement Factor (Is the set retiring soon?)
   - Discount Factor (Current market price vs. RRP)
   - Theme Factor (Historical popularity of the theme)
   - Minifig Factor (Value of included figures vs. set price)
   - Scarcity Factor (Is production limited or exclusive?)
   - Part Out Factor (Total value of individual pieces if sold separately)
   - Mismatch Factor (Is the set significantly undervalued vs. similar sets?)
